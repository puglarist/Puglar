# Puglar

This repository stores the official Puglar doctrine reference assets.

## Contents

The root contains printable and archival files, including:

- Field cards
- Handbooks and manuals
- Doctrine omnibus documents
- Historical archive ZIPs

## File Types

- `*.pdf`: printable manuals/cards and reference sheets
- `*.zip`: bundled doctrine archives

## Usage

1. Download the PDF assets you need.
2. Print according to each card or guide's instructions.
3. Keep archive ZIP files for long-term version history.

## Repository Hygiene

This repository includes binary asset defaults in `.gitattributes` and a basic
`.gitignore` for local/editor noise.
