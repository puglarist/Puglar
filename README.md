# Puglar

## Firebase web setup

This repository now includes a browser-ready Firebase initialization using your `metaverse-e158a` project config.

### Files
- `index.html`: Simple page that loads Firebase initialization.
- `src/firebase-init.js`: Initializes Firebase and Analytics with support checks.

### Install
```bash
npm install firebase
```

### Run
```bash
npm run start
```
Then open: <http://localhost:4173>
