# Puglar

This repository now includes a lightweight Python **metaverse game engine** implementation under `engine/`.

## Features
- Region graph and constrained movement.
- Player energy, health, level, and XP progression.
- Combat with weapon bonuses.
- Quest registration/assignment/completion.
- Tick-based world simulation for weather and passive recovery.
- Region snapshots for client/UI consumption.

## Run tests
```bash
python -m unittest discover -s tests -v
```
