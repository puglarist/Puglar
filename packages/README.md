# Puglar Packages

This directory provides package-style manifests for importing assets into engines or content pipelines.

- `3d/puglar-3d-pack-v1/manifest.json` — references 3D model assets.
- `maps/puglar-map-pack-v1/manifest.json` — references map assets.
- `worlds/puglar-training-world/` — starter world package that links map + 3D packs.
