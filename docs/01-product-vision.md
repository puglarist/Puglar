# Product Vision Baseline (Draft)

## Vision Statement
Create a persistent, social VR metaverse world that prioritizes embodiment, safety, creator-driven content, and low-friction cooperative play.

## Experience Pillars
1. Presence-first social interaction
2. Comfort and accessibility by default
3. Meaningful shared progression
4. Safe-by-design interactions and moderation
5. Creator extensibility with policy guardrails

## Target User Segments (Initial)
- Core VR social gamers (18-35)
- Creator-builders and world hosts
- Cooperative adventure-focused friend groups

## Candidate North-Star Metrics
- D1/D7 retention
- Average session length
- Social return rate (users returning with friends)
- User-reported comfort incidents per 1k sessions
- Moderation response SLA attainment

## Open Decisions
- Exact genre blend for the first vertical slice
- Content cadence and seasonal strategy
- Monetization policy for launch vs post-launch phases
